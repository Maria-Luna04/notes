3/31/2024 Notes
  • HTML = Computer language used to structure and write addresses used to refer to documents on the web
  • URL's = addressese usedto refer to documents and sites on the web
  • Hyptertext= Form of text in which documents can refer (link) to other documents and resources (Hyperlink)
  • Web pages -> Written in HTML. Links among web pages and other resources are what makes the web "Web like"
  • Web Pages are usually stored on servers
  • HTTP= Hypertext Transfer Protocol. Broswer makes request to server -> server sends response back to browser. = Rules how requests and responses work.
  • Browser gets web pages by communicating with browser over the internet
  • Servers= computers that have programs running on them which answers a browser's requests
  • HTTPS. S= Secure. Uses encryption to keep your data private

HTML + Programming

  • Syntax= Grammar rules of a computer language
  • Formalism= Computers take. code literally and will only do exactly what you tell it to do. Careful with spelling and punctuation.
  • Nesting = Bits of code going inside other bits of code in an orderly way.
  • HTML documents are written in text files
  • Text = anything you can type on a regular type writer. No special typography, photos, images
  • Layout of web page seen to by users is put together by web browser based on the code in the text file.

4/1/2024 Notes

  • The Job of HTML = text users will read in their browser.
  • Markup tells browsers wht the text should look like or how it's arranged. Within markup there can be refrences.
  • Markup = text that has special meaning. Every element starts and ends with a tag
  • Breaks and Empty Elements
    1. Whitespace = Spaces, tabs, and line breaks. Browser runs all whitespace together so we don't see it.
    2. br=break. Void/empty element = an HTML element with no contents and no closing tags.
  • Paragraphs
    1. p = paragraphs. Paragraphs are different than line breaks. Creates line of space in between text lines
  • Elements
    1. Strong = Bold
    2. Sub = subscript
    3. Sup = superscript
    4. em= Emphasized / Italized
    5. mark = highlighted
  • Nested Elements
    • Elements that go inside other elements. Have to follow certain Rules.
    • 2 Elements can't overlap. One must be inside the other.
    • h1 = headlines.
    • H1 is the largest and H6 is smallest

4/2/2024 Notes

  • Block Elements=Browser makes a block
  • Inline Elements = No Box. Flow with text they're used in.
  • div=Division. Divide up page into sections
    • Has invisible box around it
    • Generic container for other elements to put inside.
    • Used a lot in HTML to help layout and organize pages.
  • Lists and Implied Closed Tags
    • li Element = list. Closing Tag is optional.
    • Always used inside other elements.
  • ol = ordered list element = Letters or #'s
    • Adding Type = "a" = lowercase letters
    • Adding Type "A" = Uppercase letters
    • Adding Type "I" = Uppercase Roman Numerals
    • Adding Type "i" = Lowercase Roman Numerals
  • ul = unordered list = Bullets or dots

    • Closing tag is optional.
